# Bazhina_course_work

1. Clone the repository:
 
   ```bash
   git clone https://github.com/NataliaBazhina/Course_work.git
   
   cd Course_work
   
2. Activate venv
 
   ```bash
   python -m venv venv
   source venv/bin/activate
   or on windows
   ./Scripts/activate
   
3. Install dependencies
 
   ```bash
   pip install -r requirements.txt
   
4. Use :)
   
   ```bash
   python main.py
